# xtrack

A discord bot to track and settle your common group expenses

## Commands

| Command | Description |
|---|---|
| `$rec <amount>, <note>` | Record an expense (e.g. `$rec 12.50, lunch`) |
| `$total` | Show the total expenses in the current channel |
| `$mytotal` | Show your personal total in the current channel |
| `$settle` | Calculate who owes whom and how much |
| `$download` | Download all expense records in a spreadsheet |

**Editing & Deleting:** Supports editing and deleting a `$rec` message.

**DMs:** Can also be used to track personal expenses via DMs, same commands.
